HOW TO OPEN AND USE "Unity 2017 Tutorial - Light Probes" project IN UNITY

1. Open Unity 2017.2 or newer. Press on Open and select "Unity 2017 Tutorial - Light Probes" project.
2. Inside "Scenes" folder open "Scene_01".
3. Fix colors - go to "Edit > Project Settings > Player" and set "Color Space*" to "Linear".

-You can control the Cube using W, A, S, D or Arrow Keys. Use 'Space' to Fly.
